# Troubleshooting

| Problem | Likely Cause | Solution |
|--------|-------------|----------|
| Cannot find FigJam or Design option | Not on the Figma home page after login | Return to the main Figma dashboard and look at the top right for "FigJam" or "Design" |
| A pop-up is blocking the screen | Figma tutorial or info popup is open | Close the popup using the **X** before continuing |
| Unable to draw a shape | Wrong tool selected or nothing selected | Click the shape tool (square icon) before clicking and dragging |
| Shape is not changing color | Shape is not selected | Click the shape first, then open the color menu |
| Text is hard to read | Low contrast between text and shape | Choose darker text or lighter background for better contrast |
| Cannot type inside a shape (FigJam) | Shape not in text-edit mode | Double-click inside the shape before typing |
| Shapes are overlapping | Shapes were duplicated but not moved | Click and drag shapes apart to separate them |
| Cannot label shapes in Figma Design | Figma Design does not support direct shape text | Use the text tool and place text on top of the shape |
| Moving objects moves only part of it | Objects are not grouped | Select both objects, right-click, and click "Group selection" |
| Grouped object behaves unexpectedly | Double-click selects only one element | Click once (not double-click) to move the full group |
| Cannot connect shapes in flowchart | Wrong tool selected | Use the "Bent connector" tool from the toolbar |
| Connectors point the wrong way | Dragged from wrong connection point | Reconnect by dragging from the correct side of the shape |
| Flowchart looks messy or confusing | Shapes are not aligned or spaced properly | Drag shapes into alignment and space them evenly |
| Cannot link shapes in prototype | Trying to link to a shape instead of a frame | Create or use a frame as the destination |
| Blue connection arrow not appearing | Not in Prototype mode | Click "Prototype" in the right sidebar |
| Clicking button does nothing in preview | Frames are not properly linked | Ensure each button is connected to a frame with "On click → Navigate to" |
| Prototype opens but doesn’t switch screens | Interaction not set correctly | Reconnect and confirm action is "Navigate to" |
| Elements not inside frame | Shapes were not dragged into frame | Drag objects into the frame in the Layers panel or canvas |
| Cannot see connection lines | Not in Prototype view | Switch to "Prototype" tab to view connections |
| Can't find attributes on right sidebar of Figma Design | Your device has a smaller screen | Scroll down while hovering over the sidebar |
| Endpoint of connector in FigJam doesn't follow the shape it is attached to when moving the shape | The connector was improperly attached | Click and drag the improperly attached end to the shape it needs to attach to, ensure it snaps to the shape |
