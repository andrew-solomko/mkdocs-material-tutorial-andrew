# Glossary

| Word      | Definition |
|-----------|------------|
| Caret     | The blinking text cursor that shows where the next character will appear when you type. |
| Connector | A line or arrow that links one shape to another in a diagram. |
| FigJam    | Figma’s whiteboard tool, used for brainstorming, diagrams, and flowcharts. |
| Figma Design | Figma’s design workspace, used for creating screens, layouts, and wireframes. |
| Fill      | The inside color of a shape. |
| Flowchart | A diagram that shows the order of steps in a process. |
| Label     | Text placed inside or beside a shape to explain what it represents. |
| Shape     | A visual object, such as a rectangle or diamond, used to build diagrams. |
| Stroke    | The outline or border around a shape. |
